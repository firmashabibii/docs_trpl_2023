---
title: Login Dan Registrasi

## description:

## Login
- Jika User belum memiliki akun maka harus melakukan Registrasi terlebih dahulu
- User bisa login menggunakan Username atau Email
- Setiap User harus memiliki Username atau Password yang berbeda
- Jika User melaukan login dengan Username/Email dan Password yang salah makan akan muncul kalimat

## Registrasi
- User akan melakukan Registrasi dengan menggunakan Email
- User juga harus mengulangi Password sebanya 2 kali
- Data User akan disimpan ke database
