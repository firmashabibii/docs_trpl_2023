## Tujuan Aplikasi koslife

Aplikasi **koslife** bertujuan untuk membantu anak kost, terutama mahasiswa yang masih mengandalkan uang orang tuanya.

Tujuan:

### Mencatat Keuangan
Mempermudah dalam mencatat keuangan untuk setiap harinya.

### Melacak Keuangan
Menyediakan laporan seperti diagram chart ataupun dengan File CRUD.

### Mengelola keuangan
Membantu mengontrol mereka untuk menggunakan biaya sehari-harinya.

Aplikasi koslife juga di rancang dengan responsif yang bagus bisa di gunakan untuk di dekstop maupun mobile.
