Pengujian juga di lakukan keseluruhan terutama di CRUD, karena hampir semua menu memiliki CRUD.

## Hasil dari pengujian

- Setelah semua saya uji semua berhasil, untuk login berhasil masukn dan register berhasil menyimpan data di databse yang sudah disiapkan.
- Sama hal nya seperti category, semua berhasil disimpan di dalam database dengan menggunakan aplikasi XAMPP.
- Di dashboard juga berhasil untuk membuat grafik yang ada, disaat adanya data pemasukan/pengeluaran maupun saldo.
- Di Goals juga berhasil ketika goals dibuat maka akan tercipta atau terbentuk  Grafik Doughnut.
- Ketika di Help, data berhasil masuk kedatabse.

# Kesimpulannya

Semua data dan fungsionalitas berhasil dan berkerja dengan baik.
