## Pengembangan Kode

Pengembangan kode di Koslife adalah proses menulis program atau logika yang memungkinkan aplikasi membantu pengguna mengelola kehidupan kos mereka secara efisien.

### Tujuan:

Membuat aplikasi yang dapat membantu penghuni kos dalam mengatur keuangan, mencatat pengeluaran, dan memantau aktivitas kos sehari-hari.

### Kegiatan Utama:

- Membangun logika aplikasi di server:
Menulis kode untuk fitur seperti menambahkan, mengedit, atau menghapus catatan pemasukan dan pengeluaran.
- Membuat tampilan untuk pengguna:
Menyusun elemen visual seperti tabel pengeluaran, formulir pencatatan, atau grafik anggaran agar mudah digunakan dan intuitif.
- Menghubungkan logika server dengan tampilan:
Memastikan data pengguna, seperti catatan pengeluaran, dapat disimpan ke database dan ditampilkan kembali kapan saja melalui dashboard.

## Pengembangan Basis Data

Pengembangan basis data di Koslife adalah proses merancang dan mengelola penyimpanan data pengguna untuk mendukung fitur-fitur aplikasi.

### Tujuan:

Menyimpan semua informasi pengguna Koslife, seperti pemasukan atau pengeluaran dengan cara yang terstruktur dan mudah diakses.

### Kegiatan Utama:

- Merancang struktur penyimpanan data:
Membuat tabel seperti pengguna, pemasukan, dan pembayaran agar semua data tersimpan dengan rapi.
- Menyusun hubungan antar data:
Menghubungkan data seperti data-data tertentu
- Mengelola dan mengambil data:
Memastikan data, seperti pengeluaran dan pemasukan, dapat dimodifikasi dan diambil sesuai kebutuhan pengguna melalui aplikasi.
