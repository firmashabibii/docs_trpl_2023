# System Requirement

## Description
Aplikasi ini dirancang untuk membantu pengguna dalam mengelola keuangan pribadi, mencatat pemasukan dan pengeluaran, serta menganalisis tren keuangan.

## Functional Requirements
- Pengguna dapat mencatat pemasukan dan pengeluaran secara mudah.
- Pengguna dapat membuat dan mengelola kategori pengeluaran.
- Sistem menampilkan sisa saldo anggaran berdasarkan input pengguna.
- Sistem menyediakan analisis tren pengeluaran dalam bentuk grafik visual.

## Non-Functional Requirements
- **Responsiveness**: Aplikasi dapat digunakan di berbagai perangkat, termasuk seluler dan desktop.
- **Security**: Data keuangan pengguna dienkripsi untuk memastikan privasi dan keamanan.
