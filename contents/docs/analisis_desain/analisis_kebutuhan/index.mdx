---
title: System Requirement

## description:

## Fungsional:

Pengguna dapat mencatat pemasukan dan pengeluaran.
Pengguna dapat membuat kategori pengeluaran.
Sistem menampilkan sisa saldo anggaran berdasarkan input pengguna.
Menyediakan analisis tren pengeluaran dalam bentuk grafik.

## Non-Fungsional:

Responsif: Dapat digunakan di perangkat seluler dan desktop.
Keamanan: Data keuangan pengguna dienkripsi.
