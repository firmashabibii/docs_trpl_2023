title: 
Keterkaitan Mata Kuliah

description: 
Project ini merupakan kolaborasi dari beberapa mata kuliah, antara lain:
---
1. Basis Data: Fokus pada perancangan, implementasi, dan pengelolaan basis data menggunakan MySQL.
2. Perancangan Antarmuka Pengguna: Membahas pembuatan antarmuka aplikasi berbasis HTML, CSS, dan JavaScript.
3. Praktik Analisis Perangkat Lunak: Meliputi analisis kebutuhan dan desain sistem.
4. Pemrograman Web Lanjut: Menitikberatkan pada pengembangan fungsionalitas aplikasi menggunakan PHP.
5. Metode Model Pengembangan Perangkat Lunak: Mengimplementasikan model pengembangan perangkat lunak dalam seluruh tahapan proyek.
