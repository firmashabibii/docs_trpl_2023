title: Keterkaitan Mata Kuliah
description: Project ini merupakan kolaborasi dari beberapa mata kuliah, antara lain:

- Basis Data: Fokus pada perancangan, implementasi, dan pengelolaan basis data menggunakan MySQL.
- Perancangan Antarmuka Pengguna: Membahas pembuatan antarmuka aplikasi berbasis HTML, CSS, dan JavaScript.
- Praktik Analisis Perangkat Lunak: Meliputi analisis kebutuhan dan desain sistem.
- Pemrograman Web Lanjut: Menitikberatkan pada pengembangan fungsionalitas aplikasi menggunakan PHP.
- Metode Model Pengembangan Perangkat Lunak: Mengimplementasikan model pengembangan perangkat lunak dalam seluruh tahapan proyek.
